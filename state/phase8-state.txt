## Host firewall fix (Phase 8 unblocked external reachability)

Found during Phase 8.3 that the host iptables INPUT chain had rule #9
`REJECT --reject-with icmp-host-prohibited`, which was blocking external
TCP/443 before it reached zrok2-frontend (which binds to *:443 on the host).

Added (with explicit user approval):
  iptables -I INPUT 9 -p tcp --dport 443 -m state --state NEW -j ACCEPT
Pre-change rules archived to:
  ~/zrok-rollout/iptables-pre-phase8-20260526T162733Z.rules
Persisted via:
  netfilter-persistent save  →  /etc/iptables/rules.v4 includes the new rule
